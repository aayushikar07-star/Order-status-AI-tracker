# 🚚 Order Tracking Chatbot

A modern, animated chatbot interface designed to help users quickly track their orders using simple text commands.
This project focuses on creating an engaging user experience using smooth animations, dark UI, and interactive chatbot logic.

---

## ✨ Features

* 💬 Interactive chatbot UI (WhatsApp-style conversation)
* 🌙 Dark mode design for a modern look
* 🚚 Order tracking using Order IDs
* ⏳ Typing animation for realistic bot responses
* 🎨 Smooth animations and transitions
* 📦 Emoji-based response system
* 🖼️ Dynamic background with truck & parcel visuals
* 🎯 Parallax scrolling effect for enhanced UI experience

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling, dark theme, animations
* **JavaScript** – Chatbot logic and interactivity

---

## 🚀 How It Works

1. The user enters a message in the chatbot input field
2. Example input:

   ```
   Track order 101
   ```
3. The chatbot:

   * Extracts the order ID using pattern matching
   * Searches predefined order data
   * Returns the corresponding delivery status
4. A typing animation is displayed before the response to simulate a real chatbot

---

| Order ID | Status                         |
| -------- | ------------------------------ |
| 101      | Delivered in 2 days 🚚         |
| 102      | Shipped, arriving in 5 days 📦 |

## ▶️ How to Run the Project

1. Download or clone this repository

3. Start typing messages like:

   ```
   Track order 101
   ```
